Welkom bij de readme van Ridder Draak Kasteel.
hier wat informatie over hoe je het spel start en speelt.

Hoe start je het spel?
je kan het spel starten door op de download knop te drukken of om naar onze github te gaan en het spel te downloaden uit de games branch.
het zip bestandje pak je uit in een mapje en dan komt er een bestand uit genaamt RDK.php.
om deze te starten klik je met rechtermuisknoop op het mapje en druk op ''new terminal-venster van map''.
er opent dan een commandline, om het spel nu te starten typ je in ''php rdk.php'' en het spel start.

Hoe speel je het spel?
Ridder Draak Kasteel is niet moeilijk te spelen.
er wordt gevraagd om te kiezen uit beste 3, 5 en 7.
kies een van de opties en het spel kan beginnen.
er wordt dan gevraagd om een van de drie keuzes te maken bijvoorbeeld Draak.
de computer kiest dan zelf ook een van de drie opties en de winnaar, verliezer of gelijkspel wordt bekent gemaakt.
